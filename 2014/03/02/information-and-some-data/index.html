Information and Some Data

Mar 2nd, 2014

The progress of the game is going forward, at the moment all the algorithms for the resource fields are done, that includes prices for every level, workers your planet gain per level and of course the construction time. The formulas may be changed.

We will update with tables of the first 10 levels

The things that are working in the game

  • Building
  • Get resources per hour
  • Queue up for more buildings
  • Statistics, including Players, Planets, Colonels and General
  • Registration
  • Profile

We are also considering to add achievements in to the game, how they will work and everything else will be posted later. We also need to figure out how the galaxy aka the map will look like. In the next post we will have our message system working, some more buildings like a place there the player can train troops and hopefully the Colonel system will be working, not 100% but still.

Something we want to add to the Colonel is gear and quests. Example the Colonel goes out on a quest and finds something nice, like a new gun. Then after sometime has passed in the game we will release the option to upgrade this gun to tier 2, with the help of some kind of crystal.
