The things that are working in the game

- Building
- Get resources per hour
- Queue up for more buildings
- Statistics, including Players, Planets, Colonels and General
- Registration
- Profile

We are also considering to add achievements in to the game, how the will work and everything else will be posted later. We also need to figure out how the galaxy aka the map will look like. In the next post we will have our message system working, some more buildings like a place there the player can train troops and hopefully the Colonel system will be working, not 100% but still.

Something we want to add to the Colonel is gear and quests.
Example the Colonel goes out on a quest and finds something nice, like a new gun. Then after sometime has passed in the game we will release the option to upgrade this gun to tier 2, with the help of some kind of crystal. The only way the acquire this crystal will be via quest where you have to attack a special planet that belongs to the NPC player, were not sure yet how big of army you need but be prepared, it wont be easy!
